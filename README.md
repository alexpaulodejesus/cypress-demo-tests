# Cypress Demo Tests

Projeto de testes automatizados com Cypress, ideal para aprendizado e portfólio.

## Instalação

```bash
git clone https://github.com/alexpaulodejesus/cypress-demo-tests.git
cd cypress-demo-tests
npm install
```

## Executar testes

```bash
npx cypress open
npx cypress run
```

## Comandos Demonstrados

- `cy.visit()`
- `cy.get()`
- `cy.contains()`
- `cy.click()`
- `cy.type()`
- `cy.should()`
- `cy.select()`
- `cy.check()`, `cy.uncheck()`
